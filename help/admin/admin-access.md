# Comment demander un accès Admin fonctionnel {#how-to-request-functional-admin-access}

Connectez-vous au [!UICONTROL Identity & Access Management] portail pour [!UICONTROL Inside Adobe] demander [!UICONTROL Functional Admin Access] et suivez les étapes décrites ci-dessous.

<!-- request-functional-admin-access.xml -->

1. Connectez-vous au [portail de gestion](https://iam.corp.adobe.com) d'identité et d'accès.
2. Accédez **[!UICONTROL Accounts]** à &gt; **[!UICONTROL Access Requests]**.
3. Dans [!UICONTROL Request Access] la section, cliquez **[!UICONTROL Select an Access Request Type]** sur.
4. Indiquez si vous demandez l'accès pour vous ou quelqu'un d'autre. Si vous demandez l'accès à quelqu'un d'autre, saisissez son Adobe ID dans le champ dédié et cliquez **[!UICONTROL Find User]** sur.
5. Sélectionnez **[!UICONTROL Digital Marketing]** dans la liste Type d'accès.
6. Dans **[!UICONTROL Manage Digital Marketing Access]** la page, sélectionnez **[!UICONTROL Request Admin Console Access]** dans la liste déroulante et cliquez **[!UICONTROL Next/Submit]** sur.
7. Dans **[!UICONTROL Request Admin Console Access - Console]** la page, sélectionnez **[!UICONTROL AAM]** dans la liste déroulante et cliquez **[!UICONTROL Next/Submit]** sur.
8. Dans **[!UICONTROL Request Admin Console Access - Groups]** la fenêtre, sélectionnez **[!UICONTROL AAM Functional Admin - General Access]** dans **[!UICONTROL Available Entitlements]** la liste et remplissez une **[!UICONTROL Business Justification]** pour votre requête. Confirmez la requête en cliquant **[!UICONTROL Next/Submit]** sur.
9. Votre demande doit être approuvée pour pouvoir bénéficier d'un accès administrateur fonctionnel.
