# Mises à jour de la documentation {#documentation-updates}

Toutes les mises à jour (ajouts, suppressions et corrections) apportées au guide d'administration d'Audience Manager par date.

Pour plus d'informations sur les nouvelles fonctionnalités, les améliorations et les correctifs, reportez-vous aux [Notes de mise à jour d'Experience Cloud](https://marketing.adobe.com/resources/help/en_US/whatsnew/). Pour lire les [anciennes annonces](https://marketing.adobe.com/resources/help/en_US/whatsnew/c_legacy_releases.html) Experience Cloud, voir les notes de mise à jour précédentes. Pour [!DNL Audience Manager documentation changes, see][les mises à jour de documentation](https://docs.adobe.com/content/help/en/audience-manager/user-guide/documentation-updates/docs-2019.html).

## Mises à jour de la documentation AAM 2019 {#aam-2019-docs-updates}


| Rubrique | Description |
---------|----------|
| Macros de format HTTP | Nous avons ajouté une nouvelle macro, `REGION_ID_LIST`et trois nouveaux champs`sda`,`sda`et `sda` la `USER_LIST` macro. |
